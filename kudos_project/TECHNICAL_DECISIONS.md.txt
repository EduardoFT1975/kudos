TECHNICAL_DECISIONS.md
Decisiones Técnicas Estratégicas

1. ARQUITECTURA
- Sí a dividir en múltiples Django apps bien definidas
- Mantener Django Templates + HTMX (si es necesario) para velocidad de desarrollo

2. MAPA
- Leaflet.js + OpenStreetMap (gratis)
- Migrar a PostgreSQL + PostGIS para consultas geográficas potentes

3. IMPORTACIÓN DE CONTENIDO
- Usar APScheduler o cron jobs (sin Celery al inicio)
- Fuentes prioritarias: Wikipedia API, Overpass API (POI), UNESCO
- Siempre guardar referencia a fuente original

4. IA GENERATIVA DE CLIPS
- Fase 1: Solución local/híbrida (gTTS + moviepy + imágenes estáticas)
- Fase 2: Modelo open-source local o API económica
- Versionado de prompts para poder regenerar/actualizar clips

5. GEOLOCALIZACIÓN Y NOTIFICACIONES
- HTML5 Geolocation API (con consentimiento explícito)
- django-webpush para notificaciones
- Almacenar última ubicación conocida

6. SOCIAL LOGIN
- django-allauth (soporta Instagram, Facebook, Google, X, etc.)

7. BASE DE DATOS
- Migrar a PostgreSQL lo antes posible (obligatorio)

Recomendación fuerte: Empezar inmediatamente la separación en apps Django.

Próximo paso técnico prioritario: Refactor modular + social login.
